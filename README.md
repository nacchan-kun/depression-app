## Installation

1. **Install dependencies:**

   ```bash
   npm install
   ```

2. **Start the development server:**

   ```bash
   npm start
   ```

3. **Open your browser:**
   Navigate to [http://localhost:3000](http://localhost:3000)

## Building for Production

```bash
npm run build
```

This creates an optimized production build in the `build` folder.

## Technologies Used

- **React 18** - Frontend framework
- **CSS3** - Styling and animations
- **LocalStorage** - Data persistence

## Privacy & Data

All data (mood logs, journal entries, favorite affirmations) is stored locally in your browser. Nothing is sent to any server. Your privacy is completely protected.

## Important Note

This app is designed to provide support and coping tools, but it is **not a replacement for professional mental health care**. If you're experiencing a mental health crisis, please reach out to:

- **National Suicide Prevention Lifeline (US):** 988
- **Crisis Text Line:** Text HOME to 741741
- **Emergency Services:** 911 (US) or your local emergency number
